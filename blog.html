Question-1: What are the different ways to select an element in the DOM?

Answer:
We can select the element of the dom in different ways, examples: 
1. with Id: getElementById()
2. with classname: getElementsByClassName()
3. with tagname: getElementsByTagName()
4. with query selector: querySelector() and querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ? 

Answer:
innerHTML changes all html content or return HTML content.
innerText return the visible text or set the text.
textContent return the whole textcontent or set without html tag.

Question-3: What is event delegation in the DOM? 

Answer:
Event delegation is a kind of technique where a parent element attached 
with eventlistener and that event work on the child element.

Question-4: What is event bubbling in the DOM? 

Answer:
Event Bubbling is a mechanism of dom event handling. Where, if  the event perform on 
the parent it sprayed automatically all the elements until finishing the document .

Question-5: How do you create, add, and remove elements using JavaScript?

Answer:
We can create, add and remove a element by using JavaScript Dom manipulation.
1. To create a new element we can use document.createElement() method.
2. To add a new element we can use appendChild(), append() etc.
3. To remove an element we can use remove(), removeChild()
